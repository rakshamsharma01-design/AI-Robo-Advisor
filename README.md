# AI-Powered Robo Advisor

## Week 4 Deliverables

This project implements an AI-powered Robo Advisor using AWS services.

### Features
- Historical stock data collection using Yahoo Finance
- Logistic Regression model training
- Model parameter export (`model_params.json`)
- Amazon S3 for data and model storage
- AWS Lambda for inference
- Amazon DynamoDB for recommendation storage
- API Gateway integration
